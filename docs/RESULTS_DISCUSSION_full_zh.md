# Results & Discussion 完整章節版

## 3. Results and Discussion

### 3.1 不同栽培模式下之 N2O 排放與土壤有機碳差異

本研究比較有機茶園與慣行茶園在相同施氮量條件下之碳排放與碳匯表現。結果顯示，有機茶園與慣行茶園全年 N2O 排放量分別為 12.3 與 7.03 kg ha^-1，折算為 CO2 當量後分別為 3.36 與 1.92 t CO2e ha^-1，有機茶園之 N2O 排放量約為慣行茶園之 1.75 倍。此結果顯示，在相同氮投入條件下，有機栽培模式具有較高的 N2O 排放風險。可能原因在於有機質肥料除提供氮源外，亦增加土壤中可被微生物利用之有機碳基質，進一步促進硝化與反硝化作用，使土壤在適當水分條件下更容易產生 N2O。

另一方面，土壤深度 30 cm 之土壤有機碳含量在有機茶園與慣行茶園分別為 84.16 與 56.98 t C ha^-1，有機茶園較慣行茶園高出 27.18 t C ha^-1。此結果顯示，不同栽培模式之茶園土壤皆具有碳匯能力，而有機栽培模式在土壤有機碳累積上更為明顯。綜合來看，有機茶園雖具有較高的溫室氣體排放量，但在碳匯潛力上亦顯著優於慣行茶園，說明在碳收支評估上不能僅以 N2O 排放高低判斷制度優劣，而應同時考量土壤固碳效益。

### 3.2 雨量延遲效應與加權雨量指標

本研究以 rain_lag0 至 rain_lag7 分析降雨對茶園 N2O 排放之短期延遲影響。結果顯示，rain_lag1 與 N2O_flux 之 Pearson 相關係數最高（r = 0.2183），其次為 rain_lag4（r = 0.1639）與 rain_lag0（r = 0.1256），顯示茶園 N2O 排放並非僅受單一日雨量所控制，而是同時受到當日、前 1 日及更早期降雨條件之共同影響。此結果說明降雨對 N2O 排放之作用具有明顯延遲特性，亦即降雨事件之後，土壤含水狀態、通氣情形與微生物反應需要一定時間累積，方會形成可觀測之 N2O 排放變化。

基於此一特性，本研究進一步建立加權雨量指標 `rain_weighted`，以整合多日降雨資訊。從權重結果可知，rain_lag1 佔比最高（weight = 0.3048），rain_lag4 與 rain_lag0 分別為 0.2288 與 0.1754，而 rain_lag2 與 rain_lag5 亦具有次要貢獻。這表示茶園 N2O 排放最敏感的並非單一日降雨量，而是短期累積降雨脈衝。此現象與土壤微生物活動對水分條件之延遲反應一致，也支持茶園 N2O 排放具有事件驅動（event-driven）與脈衝型（pulse-like）特性。

### 3.3 N2O 預測模型表現

以 `rain_weighted`、`fertilizer_N`、`soil_temp_lag1`、`temp_lag1` 與 `humidity_lag1` 建立之主模型整體達顯著水準，F(6, 488) = 25.62，p < 2.2 × 10^-16，決定係數 R2 = 0.2396，調整後決定係數 Adjusted R2 = 0.2302，RMSE = 0.05434，MAE = 0.02653。雖然模型解釋力尚屬中等，但已足以顯示茶園 N2O 排放並非完全隨機，而是受到施肥、降雨脈衝與短期熱環境共同控制。就農業環境資料而言，N2O 排放本身具有高度變異、短時間尖峰與強事件依賴等特性，因此 R2 約 0.24 之模型在解釋與管理應用上仍具一定價值。

簡化模型移除土壤溫度後，其 R2 = 0.2332，Adjusted R2 = 0.2254，RMSE = 0.05457，MAE = 0.02647，與主模型之表現相當接近。此結果顯示，即使缺乏土壤感測資料，僅利用施肥量、降雨脈衝與一般氣象資訊，仍可保留多數解釋能力。這對於未來建立管理導向之低成本預測工具具有重要意義，尤其適用於未配置完整土壤監測設備的茶園場域。

### 3.4 主要驅動因子與交互作用之解釋

在主模型各係數中，`fertilizer_N` 呈顯著正向效應（β = 5.276e-05，p = 1.47e-04），顯示氮投入量愈高，茶園土壤 N2O 排放之基礎風險愈高。此結果符合一般氮循環機制，即施氮增加土壤中可供微生物硝化與反硝化使用之底物，進而提高 N2O 產生之可能性。

相較之下，`rain_weighted` 單獨主效應並未達顯著（β = 4.018e-05，p = 0.9230），顯示降雨本身不是茶園 N2O 排放之獨立驅動因子。然而，`rain_weighted:fertilizer_N` 交互作用項達高度顯著（β = 6.762e-06，p = 3.84e-08），為本模型中最重要之結果。此一結果指出，降雨的真正作用不在於單獨提升 N2O，而是在高施氮條件下放大排放反應。換言之，雨量在本系統中較接近 trigger，而施肥量則是 amplifier。當土壤中已有較高氮源，降雨造成之土壤濕潤脈衝與通氣限制，便更容易促成高排放事件。

在其他環境變數方面，`soil_temp_lag1` 呈顯著負向效應（β = -0.003166，p = 0.0444），`temp_lag1` 呈接近顯著之正向效應（β = 0.003100，p = 0.0529），而 `humidity_lag1` 未達顯著。此結果顯示，茶園 N2O 排放除受降雨與施肥交互作用控制外，亦可能受到短期土壤與空氣熱環境調節；然而相較於降雨脈衝與施氮量交互作用，這些背景因子的解釋力較為次要。

### 3.5 茶園 N2O 排放之脈衝特性

本研究進一步以 N2O_flux 第 75 百分位作為高排放事件門檻，建立 pulse detection 模型，並利用 Youden index 掃描最佳降雨觸發值。結果顯示，最佳門檻為 `rain_weighted = 4.4078`，其 sensitivity = 0.6532，specificity = 0.6927，Youden = 0.3459。此結果表示，當短期累積降雨條件超過此閾值時，高排放事件發生之機率將明顯增加。

脈衝（pulse）在本研究中的意義，是指 N2O 排放於短時間內快速升高、之後再回落之事件型排放高峰。此類高峰通常與施肥、降雨或土壤再濕潤等事件有關，反映土壤微生物在氮源充足與高含水條件下的快速反應。根據本研究結果，茶園 N2O 排放確實具有明顯之脈衝特性，且主要由降雨脈衝與施肥量共同驅動。此一結果不僅具有生態機制上的意義，也顯示茶園排放管理不能僅依固定日程施肥，而需考量施肥當期是否將遇到高雨量風險。

### 3.6 模型結果與田間碳收支之整合討論

田間監測結果指出，有機茶園在全年 N2O 排放上高於慣行茶園，但在土壤有機碳累積上亦顯著優於慣行茶園。預測模型則補充說明，茶園 N2O 排放增加的機制關鍵在於高氮供應與降雨脈衝之重疊，而非降雨本身。因此，若有機茶園因有機質肥料提供較多可分解碳源與氮源，在高濕條件下更可能出現微生物快速作用與短期高排放事件，此即有機茶園 N2O 排放較高之可能機制之一。

然而，從碳收支角度而言，有機茶園透過持續增加土壤有機碳儲量，展現出更高的碳匯能力，該效益在長期尺度上可能抵銷甚至超過其增加之 N2O 碳排風險。因此，本研究結果不應被解讀為有機栽培模式較不利於淨零，而是指出其管理上需更重視施肥與降雨之配合，亦即在維持碳匯優勢的同時，透過優化施肥時機降低脈衝型排放。

### 3.7 管理意涵與應用價值

本研究建立之模型具有明確管理應用價值。首先，模型指出高施氮條件下若遇降雨事件，茶園較易出現 N2O 高排放，因此施肥管理應避免與明顯降雨脈衝重疊。其次，簡化模型結果顯示，即使不具備土壤感測資料，仍可依據逐日氣象資料與施肥量進行基本風險評估，未來可進一步發展為茶園施肥風險預警工具。最後，pulse threshold 分析所得到之 `rain_weighted = 4.41` 可視為一項初步警戒值，作為評估高排放風險窗口之參考依據。

整體而言，本研究顯示茶園 N2O 排放之形成具有明顯的「氮源 × 降雨脈衝」交互作用機制，而不同栽培模式下之碳排與碳匯特性需整體評估。對茶園永續管理而言，未來若能進一步將施肥制度、氣象預報與土壤碳累積資訊整合，將有助於兼顧生產、減排與固碳之多重目標。
