# 口試簡報 1 頁式研究總表

## 題目

茶園土壤 N2O 排放、土壤有機碳累積與預測模型之建立

## 研究背景

- 茶樹為我國重要經濟作物，茶園需施用大量氮肥維持茶菁產量。
- 氮肥投入可能促進土壤排放 N2O，且其 100 年暖化潛勢約為 CO2 的 273 倍。
- 有機栽培可能提高土壤有機碳累積，但也可能增加 N2O 排放。
- 因此需同時評估碳排與碳匯，並建立可用於管理的預測模型。

## 研究目的

1. 比較有機與慣行茶園之 N2O 排放與土壤有機碳差異。
2. 建立茶園 N2O 預測模型，解析降雨與施肥量對排放的影響。
3. 找出高排放事件之降雨觸發門檻，作為管理參考。

## 田間研究結果

- 有機茶園全年 N2O 排放量：12.3 kg ha^-1
- 慣行茶園全年 N2O 排放量：7.03 kg ha^-1
- 折算 CO2e：有機 3.36 t ha^-1，慣行 1.92 t ha^-1
- 有機茶園 N2O 排放約為慣行茶園 1.75 倍

- 有機茶園土壤有機碳：84.16 t C ha^-1
- 慣行茶園土壤有機碳：56.98 t C ha^-1
- 有機茶園較慣行高出 27.18 t C ha^-1

## 模型資料

- 五個施肥處理：A–E
- 施氮量：0、100、200、400、600 kg N ha^-1
- 原始樣本數：505
- 有效建模樣本數：495
- 逐日氣象資料：2025 年氣象署資料

## 模型架構

主模型：

`N2O_flux ~ rain_weighted * fertilizer_N + soil_temp_lag1 + temp_lag1 + humidity_lag1`

關鍵概念：

- `rain_weighted`：多日加權雨量指標
- `fertilizer_N`：施氮量
- `rain_weighted × fertilizer_N`：降雨與施肥量交互作用

## 主要模型結果

- 主模型 R2 = 0.2396
- Adjusted R2 = 0.2302
- RMSE = 0.05434
- MAE = 0.02653

關鍵係數：

- `fertilizer_N` 顯著正向
- `rain_weighted` 單獨不顯著
- `rain_weighted × fertilizer_N` 高度顯著

## 核心解釋

- 雨量本身不是單獨驅動因子
- 高施氮條件下，降雨會顯著放大 N2O 排放
- 茶園 N2O 排放具有明顯之降雨脈衝特性

一句話總結：

「雨量是 trigger，施肥量是 amplifier」

## 脈衝與門檻

- 脈衝：N2O 在短時間內快速升高、之後回落的高排放事件
- 最佳降雨觸發門檻：
  - `rain_weighted = 4.4078`
- 對應表現：
  - sensitivity = 0.6532
  - specificity = 0.6927
  - Youden = 0.3459

## 結論

- 有機茶園 N2O 排放較高，但土壤有機碳累積也顯著較高。
- 茶園 N2O 排放主要受施肥量與降雨脈衝交互作用控制。
- 若能避免高施氮條件與降雨事件重疊，可降低高排放風險。
- 有機栽培雖增加部分 N2O 碳排，但在提升碳匯與土地永續方面仍具有明顯優勢。

## 管理應用

- 作為施肥避雨管理依據
- 作為高排放風險日預警工具基礎
- 可進一步整合為茶園碳收支與低碳管理決策架構
