# N2O 模型結果草稿

本研究整合五種施肥處理 A-E 之 N2O 通量資料與 2025 年逐日氣象資料建立預測模型。A、B、C、D、E 分別對應 0、100、200、400、600 kg N ha^-1。原始樣本共 505 筆，經 lag 變數與必要自變數完整性篩選後，最終納入分析之有效樣本為 495 筆。

## Rainfall lag 權重結果

以 N2O_flux 與 rain_lag0-rain_lag7 之 Pearson 相關係數絕對值建立 rain_weighted 指標。結果顯示，rain_lag1 與 N2O_flux 之相關性最高，r = 0.2183，p = 9.35e-07，權重為 0.3048；其次為 rain_lag4，r = 0.1639，p = 2.49e-04，權重為 0.2288；再者為 rain_lag0，r = 0.1256，p = 0.0051，權重為 0.1754。rain_lag2 仍呈弱但達邊界顯著之正相關，r = 0.0882，p = 0.0500。整體而言，N2O 排放不僅受前 1 日降雨影響，亦帶有當日與前 4 日降雨之累積訊號，支持多日降雨脈衝而非單日雨量驅動之機制。

## 主模型結果

主模型設定為：

`N2O_flux ~ rain_weighted * fertilizer_N + soil_temp_lag1 + temp_lag1 + humidity_lag1`

模型整體達顯著水準，F(6, 488) = 25.62，p < 2.2e-16；決定係數 R2 = 0.2396，調整後決定係數 Adjusted R2 = 0.2302，RMSE = 0.05434，MAE = 0.02653。這表示目前納入之降雨脈衝、施肥量與環境條件可解釋約 24% 的 N2O 排放變異。

在各解釋變數中，施肥量主效應呈顯著正向影響，beta = 5.276e-05，p = 1.47e-04，顯示在控制其他因子後，施氮量越高，N2O 基礎排放風險越高。土壤溫度前一日變數亦達顯著，beta = -0.003166，p = 0.0444；前一日氣溫則接近顯著，beta = 0.003100，p = 0.0529。相對地，rain_weighted 單獨主效應未達顯著，beta = 4.02e-05，p = 0.9230，humidity_lag1 亦未達顯著，beta = 1.94e-04，p = 0.4272。

最重要的是，rain_weighted 與 fertilizer_N 的交互作用達高度顯著，beta = 6.762e-06，t = 5.587，p = 3.84e-08。此結果顯示，降雨脈衝本身不是獨立驅動因子，但在高氮條件下會顯著放大 N2O 排放反應。也就是說，雨量在本系統中的角色較接近 trigger，而施肥量則是 amplifier。

## 簡化模型比較

為評估在不納入土壤資料情況下之可預測性，本研究另建立簡化模型：

`N2O_flux ~ rain_weighted * fertilizer_N + temp_lag1 + humidity_lag1`

簡化模型之 R2 = 0.2332，Adjusted R2 = 0.2254，RMSE = 0.05457，MAE = 0.02647。與主模型相比，模型解釋力僅小幅下降，顯示即使不使用土壤感測資料，僅憑降雨脈衝、施肥量與一般氣象因子，仍可保留大部分預測能力。這代表簡化模型具有更高的管理應用潛力，尤其適合缺乏即時土壤監測設備之場域。

## Pulse detection 與觸發門檻

本研究以 N2O_flux 第 75 百分位作為 pulse 定義，並進一步以 rain_weighted 掃描降雨觸發門檻。最佳 rainfall trigger threshold 為 4.4078，其 sensitivity = 0.6532，specificity = 0.6927，Youden index = 0.3459。對應之分類結果為 TP = 81、TN = 257、FP = 114、FN = 43。

此結果表示，當 rain_weighted 高於約 4.41 時，N2O 高排放事件發生機率明顯增加。雖然該門檻無法完全區分所有 pulse 與非 pulse 事件，但已可作為實務上的風險警戒值，用於預警高施肥條件下的 N2O 排放窗口。

## 結果詮釋

綜合而言，本研究的主要發現有三點。第一，N2O 排放對降雨具有多日延遲反應，其中前 1 日、前 4 日及當日降雨之訊號最為明顯。第二，降雨脈衝單獨主效應不顯著，但與施肥量存在強烈交互作用，支持「高氮供應下，降雨事件驅動 N2O pulse」之機制。第三，即使移除土壤溫度資料，簡化模型仍保有與主模型相近之解釋力，顯示該架構可進一步發展為管理導向之預測工具。
